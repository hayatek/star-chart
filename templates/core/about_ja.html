このWebアプリについて


このアプリはGitHub GraphQL API v4を使用してGitHub上から対象リポジトリのスター数を取得して比較するアプリです。


使い方

①トップページのフォームに比較したいリポジトリを入力します。

  • ※1 リポジトリは2つ以上の入力が必要です。
  • ※2 入力文字列をあいまい検索します。

②「スター数を比較する」ボタンを押す。

③①で入力されたリポジトリのGitHub上のスター数をグラフで表示します。

  • Daily :当日0:00時点のスター
  • Monthly:毎月1日時点のスター数
