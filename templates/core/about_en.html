About this app


This app gets the number of stars of the repository via GitHub API and displays it in charts.

Instructions

1) Input repository name into the form on the Top page.

  • Note: 1.Multiple repositories should be input.
  • Note: 2.Fuzzy search is performed using input character strings.

2) Push "Compare" button.

3) Get the number of stars via GitHub API v4 and make charts.

  • Daily:stars as of 0:00
  • Monthly:stars as of the beginning of month
