# star-chart
フォームに入力されたGitHubリポジトリのスター数をGitHub APIより取得してグラフで比較するWebアプリです。
Celeryを使ってGithubから毎日1回データを取得して日次のグラフを作成するのと、月次では毎月1日のデータを蓄積したグラフを作成します。

## Web
https://django-starchart.herokuapp.com/

## 使用技術
- Django2
- Bootstrap4
- GraphQL
- chart.js
- Celery
- Docker
- CircleCi

## 本番環境
